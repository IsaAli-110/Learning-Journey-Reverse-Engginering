# Log 01: Dasar Arsitektur Komputer

## Deskripsi
Mempelajari siklus *Fetch-Decode-Execute* pada CPU dan peran register sebagai tempat penyimpanan data sementara.

## Poin Penting
1. **Register**: Tempat penyimpanan data tercepat (EAX, EBX, ECX, EDX).
2. **Stack Pointer (ESP/RSP)**: Penunjuk alamat memori teratas dalam stack.
3. **Instruksi Dasar**: `MOV` (memindahkan data), `PUSH`/`POP` (manipulasi stack).
