# Log 03: Disassembly Basics

## Deskripsi
Proses mengubah kode mesin menjadi bahasa Assembly agar bisa dibaca manusia.

## Alat
* Ghidra / IDA Pro / Cutter.

## Langkah Analisis
1. Load file ke dalam disassembler.
2. Identifikasi *Function Entry Point*.
3. Analisis *Control Flow Graph* (CFG) untuk melihat alur logika program (percabangan `if/else`).
